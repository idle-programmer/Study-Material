*Difference b/w Django vs Fast API vs Flask
->
Django: Full Web Framework includes admin, ORM, auth, template.Uses WSGI
FastAPI: API Framework,high-performance, async bydefault,used for microservice. Uses ASGI
Flask: Lightweight and flexible micro-framework.Uses WSGI

*⁠Which to use Django or Fast API?
->
Django: Business-heavy projects
FastAPI: High Performance or microservice architecture, High-traffic public APIs

*⁠What is CSRF?
->
Security Attact where a malicious site forces a logged-in user to perform 
unintended actions on a trustes site using their authenticated session
Example:
User logs into a trusted site (e.g. bank.com)
Session cookie is stored in the browser
User visits a malicious website
Malicious site sends a hidden request to bank.com
Browser automatically attaches cookies
Server thinks the request is genuine.

*Have you faced memory leak issue and how did u solve?
-> Memory leaks can happen due to unclosed DB connections, infinite object references, or improper caching.
Tools & Fixes: tracemalloc, objgraph, psutil, Fix circular references, Use generators instead of loading big lists,
Close DB/file connections

*Components of jwt token? (total 3 component)
-> Header: Token type (JWT), Algorithm (HS256, RS256)
Payload: UserID, Roles, Expiry, Issued time
Signature: secret_key

*How do you handle refresh token(RT)?
-> Access tokens are short-lived, and RT are long-lived. When access token expires, 
the client sends the RT to get a new access token. Blacklist RT on logout
